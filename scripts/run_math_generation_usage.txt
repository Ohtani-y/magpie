# Usage Guide for run_math_generation.sh with Command Line Arguments

## Interactive Mode (default)
./run_math_generation.sh

## Non-Interactive Mode with Arguments

### Format:
./run_math_generation.sh <model_choice> <mode_choice> [domain_choice] [custom_count]

### Examples:

1. Generate algebra problems using DeepSeek-R1-Distill-Qwen-32B (Single domain mode):
   ./run_math_generation.sh 1 1 1
   # Model 1 = DeepSeek-R1-Distill-Qwen-32B
   # Mode 1 = Single Domain
   # Domain 1 = Algebra (uses default 10K problems)

2. Generate all domains using Qwen2.5-Math-72B:
   ./run_math_generation.sh 5 2
   # Model 5 = Qwen2.5-Math-72B-Instruct
   # Mode 2 = All Domains (44K problems total)

3. Generate 5000 calculus problems using DeepSeek-R1-Distill-Llama-70B:
   ./run_math_generation.sh 2 1 2 5000
   # Model 2 = DeepSeek-R1-Distill-Llama-70B
   # Mode 1 = Single Domain
   # Domain 2 = Calculus
   # 5000 = Custom problem count

### Model Choices:
1 = DeepSeek-R1-Distill-Qwen-32B
2 = DeepSeek-R1-Distill-Llama-70B
3 = DeepSeek-R1-0528-FP4
4 = Gemma-3-27B-it
5 = Qwen2.5-Math-72B-Instruct
6 = Qwen2.5-Coder-32B-Instruct

### Mode Choices:
1 = Single Domain
2 = All Domains
3 = Custom Selection (not supported in non-interactive mode)

### Domain Choices (for mode 1):
1 = Algebra (10K default)
2 = Calculus (10K default)
3 = Geometry (6K default)
4 = Statistics (6K default)
5 = Number Theory (4K default)
6 = Discrete Mathematics (8K default)
